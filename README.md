## 📋 Project Overview

An intelligent, affordable anti-phishing email filtering system designed specifically for Small and Medium Enterprises (SMEs) in Zambia.

### Key Features
- Machine Learning model (Random Forest + TF-IDF) with **95.83% accuracy**
- Real-time email scanning via SMTP proxy (port 2525)
- User-friendly web dashboard (Flask)
- Quarantine page with feedback system (continuous learning)
- Live Monitor
- Performance Reports with charts
